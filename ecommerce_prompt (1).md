# Clothing eCommerce Website -- Full System Prompt

## 🎯 Objective

Build a full-stack, scalable clothing eCommerce website with Admin,
User, and Guest roles. The system should handle orders, records,
inventory, and provide a smooth shopping experience.

------------------------------------------------------------------------

## 👥 User Roles

### Guest

-   Browse products
-   Add to cart
-   Checkout without login

### User

-   Signup/Login
-   Save addresses
-   Order history
-   Wishlist

### Admin

-   Dashboard (sales, orders)
-   Manage products
-   Manage users
-   Handle orders & inventory

------------------------------------------------------------------------

## 🧭 Full User Flow

### 1. Landing Page

-   Hero banner
-   Categories
-   Featured products

### 2. Product Browsing

-   Filters (size, price, color)
-   Search bar

### 3. Product Details

-   Images
-   Description
-   Add to cart

### 4. Cart

-   Edit/remove items
-   Apply coupons

### 5. Checkout

-   Shipping info
-   Payment (Stripe / COD)
-   Order confirmation

------------------------------------------------------------------------
